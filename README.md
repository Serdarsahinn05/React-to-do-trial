# 📝 React + TypeScript To-Do List
![React](https://img.shields.io/badge/react-%2320232a.svg?style=for-the-badge&logo=react&logoColor=%2361DAFB)
![TypeScript](https://img.shields.io/badge/typescript-%23007ACC.svg?style=for-the-badge&logo=typescript&logoColor=white)

Bu proje, modern web geliştirme pratiklerini uygulamak amacıyla geliştirilmiş, **TypeScript** tabanlı bir yapılacaklar listesi uygulamasıdır. Proje yapılandırması için **Vite** kullanılmıştır.

## 🚀 Özellikler

* **Tip Güvenliği:** Tüm veri yapıları ve bileşenler **TypeScript** ile tanımlanmıştır.
* **Kalıcı Veri (Persistence):** `localStorage` entegrasyonu sayesinde tarayıcı yenilense bile görevleriniz kaybolmaz.
* **Dinamik Yönetim:** Görev ekleme, silme ve tamamlandı olarak işaretleme fonksiyonları mevcuttur.
* **Modern Geliştirme:** Proje, **JetBrains** ekosistemi (Webstorm) üzerinde geliştirilmiştir.

## 🛠️ Teknolojiler

* **Framework:** React (TSX)
* **Dil:** TypeScript
* **Build Aracı:** Vite
* **Paket Yöneticisi:** npm

## 📦 Kurulum ve Çalıştırma

Projeyi yerel ortamınızda ayağa kaldırmak için şu adımları izleyin:

1. **Depoyu klonlayın:**
```bash
   git clone <https://github.com/Serdarsahinn05/React-to-do-trial>
  ```

Bağımlılıkları yükleyin:

```Bash
npm install
```
Geliştirme sunucusunu başlatın:
```Bash
npm run dev
```

📂 Proje Yapısı
```Plaintext
src/
├── App.tsx          # Ana uygulama mantığı
├── ToDoList.tsx     # Liste ve görev yönetimi bileşeni
├── main.tsx         # Giriş noktası
└── index.css        # Global stiller
```
